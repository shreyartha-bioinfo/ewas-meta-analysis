# Results

Rendered tutorial output and example result files are stored here.

## Files

| File | Description |
|------|-------------|
| `ewas_metaanalysis_tutorial.html` | Fully rendered tutorial with all plots and outputs |

## Generated at runtime

When you run the tutorial, these files are also produced in the working
directory:

| File | Description |
|------|-------------|
| `ewas_meta_results.csv` | Full CpG-level meta-analysis results (FE + RE + heterogeneity + FDR) |
| `ewas_meta_dmr_results.csv` | Differentially methylated regions (Stouffer combination) |

These are gitignored — they are regenerated each time you run the tutorial.
